שרה דנינו שטיין

סבתא שרה (פרידה) נולדה ב 6.11.1945 בוילנה בעיר ליטא לזוג הורים ניצולי שואה.
אביה חיים שטיין איבד את אשתו ושלושת ילדיו במלחמה ונותר עם בן אחד, אברהם.
אמה מרים בראוודה איבדה את בעלה במלחמה ונותרה עם ילדה אחת, טובה.
חיים ומרים הכירו אחרי המלחמה, התחתנו וסבתי נולדה.
במשך כ – 3 שנים עברו מארץ לארץ דרך מעברי הגבול, מתוכם שנתיים גרו בניס במחנה אוהלים.
בינואר 1948 עלתה סבתי עם הוריה ואחותה לארץ ישראל באוניית "עצמאות". בהגיעם לארץ, התגוררו במשך מספר חודשים אצל אחות האב, מרים שגרה בתל אביב.
לאחר מכן, עברו לנתניה וגרו בשכירות ברחוב גבע בעיר ומשם עברו לבית שרכשו בשיכון ותיקים (שכונה בנתניה).
את רוב שנות ילדותה העבירה בשכונה זו. שם גם הכירה בגיל 7 את סבא אלי שגר בבית מולה.
סבתי הייתה ילדה מאד שובבה, עקשנית ובעלת כושר מנהיגות. בבית הספר היסודי הצטיינה בשיעורי הספורט אותם מאד אהבה. בהיותה בת 11 עברה המשפחה לבית חדש ברחוב סמילנסקי בעיר.
בסיום כיתה ט' החליטה כי אינה רוצה ללמוד יותר ולכן עזבה את הלימודים ופנתה ללימודי ספרות.
בשל כישרונה הרב לעיצוב שיער ואהבתה למקצוע זה. לימודי הספרות ערכו כחצי שנה ומאחר והצטיינה בלימודים אלו, נבחרה לעבוד במספרת בית הספר לספרות.
כעבור זמן, עברה לעבוד במספרה של הדוד בנימין כעצמאית ובהמשך הפכה להיות ספרית שכירה. מגיל 16 עד לגיל הפנסיה עבדה סבתי כספרית.
ב 25.8.64 בגיל 19 התחתנה סבתי עם סבי אלי ולאחר כשנה הפכו להורים. נולדו להם 3 בנות רונית, ריקי וקרן, 10 נכדים ונין (חמסה חמסה).
ב 6.1.98 נפטר סבי ממחלה קשה כשהוא בן 53.
למרות הקושי, סבתי דואגת לארח את כל המשפחה בכל יום שישי ושבת לארוחות משפחתיות ושומרת על איחוד המשפחה. סבתי ידועה כבשלנית מעולה. ביתה תמיד פתוח וידועה בהכנסת האורחים המיוחדת שלה.
סבתי בכל שנותיה דאגה ודואגת למשפחתה ובקשר יום יומי עם כולם וזה הדבר הכי חשוב לה בעולם!
לאחר יציאתה לפנסיה נוהגת סבתי לצאת בבוקר לבית קפה ליד הבית ולהיפגש עם חברותיה.

שאלה 1 - הבדלים בין web 1.0, web 2.0 וweb 3.0

Web 1.0 הוא לקריאה בלבד. מציג נתונים שטוחים. האינטראקציה מוגבלת בין האתרים למשתמשים.

Web 1.0 משמש את המשתמשים לקבל מידע באופן פסיבי מבלי אפשרות לפרסם ביקורת, הערות או משוב. חברות מספקות תוכן שאנשים צורכים (Client-Server).

2.0 הוא writable – ניתן גם לכתיבה. מאפשר אינטראקציה בין משתמשי האינטרנט והאתרים ובינם לבין עצמם. 0 הוא writable – ניתן גם לכתיבה. מאפשר אינטראקציה בין משתמשי האינטרנט והאתרים ובינם לבין עצמם.

מעודד שיתוף פעולה ומידע. אנשים מפרסמים תוכן שאנשים אחרים יכולים לצרוך גם כן וחברות בונות אפליקציות/אתרים שמאפשרות לאנשים לפרסם תוכן.

Web 3.0 הוא גם executable – ניתן להפעלה. הפעלת רשת אינטרנט מורחבת עם יישומים דינמיים, שירותים אינטראקטיביים ואינטראקציה "מכונה למכונה".

רשת סמנטית המתייחסת לעתיד. בweb 3.0 מחשבים יכולים לפרש מידע כמו בני אדם, לייצר ולהפיץ בצורה חכמה תוכן שימושי שמותאם לצורכי המשתמשים.

אנשים בונים אפליקציות שלאנשים אחרים יכולה להיות אינטראקציה איתן וחברות בונות פלטפורמות שמאפשרות לאנשים לפרסם שירותים המתבסס על שימוש של אנשים וקשר בין אנשים.

Web 1.0 הוא hypertext/CGI web סטטים בסיסיים המיועדים לשיתוף מידע לעומת web 2.0 הוא יותר קהילתי, קיימות אפליקציות ואתרים המקשרים בין אנשים ו web 3.0 הוא סמנטי, למכונות.



שאלה 2 - חיפוש שאילתא - תופעות לוואי בתרופות

Web 1.0 המשתמש צריך לדעת את הurl של הדף אותו הוא מחפש עבור השאילתא. במידה ולא יודע את הurl, מחפש את הדף ברשימת דפים. התוצאות לא מדויקות ולכן יכול להיות שיקבל תוצאות לא רלוונטיות

Web 2.0 המשתמש מקליד את השאילתא אותה הוא מעוניין לחפש במנוע חיפוש, דפים בנושא השאילתא ודפים שמתוייגים במילים או בתמונות של אחת ממילות השאילתא מתקבלים בנוסף, דפים שמשתמשים אחרים העלו ותגובותיהם גם מגיעים לתוצאות. יכולים לא לקבל את התשובה עצמה.

Web 3.0 המשתמש מקליד את השאילתא אותה הוא מעוניין לחפש במנוע חיפוש, נקבל את התוצאות הטובות ביותר הקיימות ברשת המכילים דפים המכילים את מילות השאילתא, תגיות קשורות, עמודים בעלי מילים נרדפות עמודים שקשורים לעמודים שחיפשו
